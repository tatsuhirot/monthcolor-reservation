# SalonBoard セッション更新マニュアル

**対象:** 週1回、SalonBoardのセッションを更新する作業  
**所要時間:** 初回15分 / 2回目以降3分

---

## 初回セットアップ（最初の1回だけ）

### 手順1：Node.js をインストール

1. ブラウザで https://nodejs.org を開く
2. 「LTS」と書かれた緑のボタンをクリックしてダウンロード
3. ダウンロードしたファイルを開いて「次へ」を繰り返してインストール完了

> ✅ Windowsでもmacでも同じ手順です

---

### 手順2：フォルダを受け取る

担当者から以下のフォルダを受け取ってください：

```
hpb-calendar/
  ├── セッション更新.bat        ← Windows用
  ├── セッション更新.command    ← Mac用
  ├── update-session.js
  ├── .env                      ← 設定ファイル（編集不要）
  └── package.json
```

フォルダはどこに置いてもOKです（デスクトップ推奨）。

---

### 手順3：Macの方のみ — 実行許可の設定

Macの場合、初回のみターミナルで以下のコマンドを実行してください：

1. `セッション更新.command` を**右クリック** → 「情報を見る」
2. 「ターミナルで開く許可」を確認
   
または、ターミナルを開いて以下を貼り付けてEnter：
```
chmod +x ~/Desktop/hpb-calendar/セッション更新.command
```

---

## 毎週の作業（2〜3分）

### Windowsの方

1. `セッション更新.bat` を**ダブルクリック**
2. 黒い画面が開く
3. `SalonBoardのパスワードを入力してください:` と表示されたらパスワードを入力（画面には表示されません）
4. Enterを押す
5. Firefoxが自動で開いてログイン → 自動で閉じる
6. `✅ 完了！` と表示されたらEnterを押して画面を閉じる

---

### Macの方

1. `セッション更新.command` を**ダブルクリック**
2. ターミナルが開く
3. `SalonBoardのパスワードを入力してください:` と表示されたらパスワードを入力
4. Enterを押す
5. Firefoxが自動で開いてログイン → 自動で閉じる
6. `✅ 完了！` と表示されたらEnterを押して画面を閉じる

---

## よくあるエラー

| エラーメッセージ | 対処法 |
|---|---|
| `Node.js がインストールされていません` | 手順1からやり直す |
| `ログイン失敗` | パスワードを確認して再実行 |
| `アップロード失敗` | インターネット接続を確認して再実行 |
| Firefoxが開かない | 少し待ってから再実行 |

---

## 更新のタイミング

- **毎週1回**（月曜日推奨）
- セッションの有効期限は約2週間です
- 更新を忘れると予約の自動同期が止まる場合があります
